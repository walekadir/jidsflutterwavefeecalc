Jid's Flutterwave Fee Calculator (v3)

A high-precision, PWA-enabled fee estimator for Nigerian businesses and freelancers. 

 What's New in v3
- Reverse Mode: Calculate exactly what to charge to receive a specific net amount.
- Live FX Rates: Real-time NGN conversion for USD, GBP, and EUR settlements.
- PWA Ready: "Add to Home Screen" support for offline access.
- PDF Export: Generate professional transaction breakdowns for your clients.

Tech Stack
- Frontend: Vanilla JS, CSS3 (Glassmorphism), HTML5.
- API: [Open Exchange Rates](https://open.er-api.com/) for live FX.
- Service Workers: For offline caching and PWA functionality.
